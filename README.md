# Folder Manager

A collection of scripts for organizing and restoring files.

## Scripts

### file_org.py

A Python script that automatically organizes files into category folders based on file extension.

**Features:**
- Categorizes files into Images, Docs, PDF, Code, etc.
- Dry-run mode to preview changes
- Recursive organization
- Generates a log file for restoration
- Handles duplicate filenames

**Usage:**
```bash
# Organize files in Downloads folder
python file_org.py --src ~/Downloads

# Dry run (preview only)
python file_org.py --src ~/Downloads --dry-run

# Organize to specific destination
python file_org.py --src ~/Downloads --dest ~/Organized

# Recursive organization
python file_org.py --src ~/Downloads --recursive
```

**Categories:**
- Images: jpg, jpeg, png, gif, bmp, svg, webp, heic
- PDF: pdf
- Docs: doc, docx, txt, rtf, odt, md
- Spreadsheets: xls, xlsx, csv
- Slides: ppt, pptx, key
- Audio: mp3, wav, aac, flac, m4a
- Video: mp4, mov, avi, mkv, webm
- Archives: zip, rar, 7z, tar, gz
- Code: py, js, html, css, json, java, cpp, c, sh

---

### restore_files.sh

A Bash script that restores files to their original locations using a log file generated by file_org.py.

**Features:**
- Reads the log file created by file_org.py
- Moves files back to their original locations
- Creates directories if needed
- Handles missing files gracefully

**Usage:**
```bash
# Restore files using a specific log file
./restore_files.sh organize_log_20260718_193926.txt

# Restore files with full path
./restore_files.sh ~/Organized/organize_log_20260718_193926.txt
```

**Log File Format:**
```
source/path/file.txt  ->  destination/path/file.txt
```

---

## Workflow

1. **Organize files:**
   ```bash
   python file_org.py --src ~/Downloads --dest ~/Organized
   ```

2. **Restore files (if needed):**
   ```bash
   ./restore_files.sh ~/Organized/organize_log_20260718_193926.txt
   ```

## Requirements

- Python 3.6+
- Bash shell
